AltaPay and Demandware plug-in

AltaPay has made it much easier for you as merchant/developer to receive secure payments in your Demandware web shop. AltaPay is fully integrated with Demandware via a plug-in (cartridge).

Get started:

  1. Sign a contract with AltaPay
  2. Install int_altapay cartridge
  3. Import ‘Metadata_AltaPay.xml’
  4. Conduct tests of the payment flow before going live. AltaPay provides a test setup for accepting card and alternative payments.
